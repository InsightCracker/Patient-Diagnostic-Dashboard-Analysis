# 🏥 Patient Diagnostic Dashboard Analysis  

## 📌 Project Overview  
The Patient Diagnostic Dashboard provides a comprehensive analysis of patient health data, focusing on risk classification, visit behavior, admission trends, and disease prevalence.

This project transforms raw healthcare data into actionable insights that support data-driven decision-making, resource optimization, and improved patient outcomes.


## 🎯 Business Problem  
Healthcare providers often face challenges such as:
- Late identification of high-risk patients  
- Poor tracking of revisit patterns  
- Limited insight into disease trends  
- Inefficient resource allocation  

This dashboard solves these issues by offering a centralized analytical view of patient data.

## 🔍 Key Insights  

### 📈 Patient Risk & Trends  
- Over half of patients are high-risk  
- Peak periods occur in February and October  

### 👨‍⚕️ Age-Based Analysis  
- 61–85 years: Highest risk severity  
- 19–35 years: Higher admission rates  

### 🔁 Visit Behavior  
- Most patients revisit within 6–12 months  
- Indicates irregular follow-up patterns  

### 🦠 Disease & Symptoms  
- Most common condition: Asthma  
- Key symptoms linked to admission:
  - Fever  
  - Cough  
  - Joint pain  
  - Diarrhea  

### 💊 Medication Trends  
- Aceclofenac  
- ACT (Antimalarial)  
- Amoxicillin  


## 🧪 Data Sources  
- Patient demographics  
- Visit records  
- Symptom logs  
- Diagnostic test results  
- Medication data  


## ⚙️ Methodology  

### Data Cleaning  
- Removed duplicates  
- Handled missing values  

### Data Transformation  
- Risk classification (Low, Medium, High)  
- Aggregation by age, time, and risk level  

### Analysis Techniques  
- Trend analysis  
- Distribution analysis  
- Correlation analysis  

### Visualization  
- Line charts  
- Bar charts  
- Donut charts
